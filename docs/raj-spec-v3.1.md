# V3.1 Minor enhancements and Bug fixes

## Enhancement: Deletion of a rental item that is already rented

Deletion should be allowed only if the item is not already rented (checked out) or if it is in someones cart.

## Bug fix: Cart Badge count error

When I create a new user and sign-in, i see a cart's badge count of 1 even though the cart is empty. Fix this and test it.

## Bug fix: multiple image view support

When an item has multiple images, only one can be seen. Need a way to indicate that there are multiple images and need a way to scroll through all images.

## Enhancement: Improve messaging when item not available

When an item is already rented, we are displaying a message. However that message does not specify the overlapping days. It will help to show the dates when that item is already reserved so that renter can reserve for remaining days.
